# 🌉 Surendra Vattikuti - Portfolio

A beautiful, interactive portfolio website featuring a 3D Golden Gate Bridge animation with a driving experience. Built with React, TypeScript, Three.js, and GSAP.

## ✨ Features

- **3D Golden Gate Bridge Scene**: Interactive 3D visualization of the iconic San Francisco landmark
- - **Immersive Driving Animation**: Experience a smooth camera movement as if driving across the bridge
  - - **San Francisco Theme**: Skyline, sunset effects, and tech-inspired design
    - - **Responsive Design**: Works seamlessly on desktop and tablet devices
      - - **Smooth Animations**: GSAP-powered animations for fluid interactions
        - - **Modern Tech Stack**: React, TypeScript, Three.js, and Tailwind CSS
         
          - ## 🛠 Tech Stack
         
          - - **React 18+**: UI framework
            - - **TypeScript**: Type-safe JavaScript
              - - **Three.js**: 3D graphics library
                - - **GSAP**: Animation library
                  - - **Vite**: Build tool and dev server
                    - - **Tailwind CSS**: Styling

                        ## 🚀 Getting Started

                        ### Prerequisites

                        - Node.js 16+ and npm or yarn
                        - - Git
                         
                          - ### Installation
                         
                          - 1. Clone the repository:
                            2. ```bash
                               git clone https://github.com/surendravattikuti01-coder/my-portfolio.git
                               cd my-portfolio
                               ```

                               2. Install dependencies:
                               3. ```bash
                                  npm install
                                  # or
                                  yarn install
                                  ```

                                  3. Start the development server:
                                  4. ```bash
                                     npm run dev
                                     # or
                                     yarn dev
                                     ```

                                     4. Open your browser and navigate to `http://localhost:5173`
                                    
                                     5. ### Build for Production
                                    
                                     6. ```bash
                                        npm run build
                                        # or
                                        yarn build
                                        ```

                                        ## 🎨 Customization

                                        ### Change Personal Information

                                        Edit `src/App.tsx` to update:
                                        - Your name and title
                                        - - About section
                                          - - Links to social media
                                            - - Portfolio projects
                                             
                                              - ### Modify 3D Scene
                                             
                                              - Edit `src/components/GoldenGate3D.tsx` to:
                                              - - Adjust bridge geometry and colors
                                                - - Change lighting and camera settings
                                                  - - Modify animation speed and effects
                                                    - - Add more 3D elements
                                                     
                                                      - ### Update Styling
                                                     
                                                      - - Modify `src/styles/App.css` for global styles
                                                        - - Use Tailwind CSS classes in components
                                                          - - Customize colors and animations
